### 项目目录

``` lua
code-generator
├── generator -- 后端
└── ui        -- 前端
```

### 项目运行

```bash
# 前端
pnpm install
pnpm run dev
```

后端放入 IDEA 中，下载依赖后运行

```bash
# jar 包内包含前端与后端，可直接通过命令运行使用
java -jar generator.jar
```
